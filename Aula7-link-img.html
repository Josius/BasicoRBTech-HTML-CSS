Acima, como estamos usando um site externo, é interessante add ' target="_blank"' para abrir uma em uma nova aba o link. Boas práticas. Outra tag interessante é a 'title=""' que permite, quando colocarmos a seta do mouse sobre o link, aparece a mensagem add na tag title. Melhora a navegabilidade no site. Por padrão, a imagem é exibida no tam original, mas é possível alterar, ver abaixo desta img

Imagem alterada com o atributo 'width="100"' -> Mas isso é considerado errado, o correto é usar CSS para alterar o tamanho das imagens. Isso porque desta forma estaremos carregando uma img grande que pode deixar o carregamento mais lento sendo exibida em um tamanho reduzido. O ideal é redimensionar a imagem antes de upá-la.

É obrigatório ter a tag 'alt="icone html"', seu uso pode ser quando a imagem estiver carregando ela vai aparecer no local da img, ou quando for especionar elemento ou para ser usada nos motores de busca, como google.
